A simple Python function to calculate Return on Investment.


def calculate_roi(investment, earnings):
    """Calculates the Return on Investment."""
    net_profit = earnings - investment
    if investment == 0:
        return 0.0
    roi = (net_profit / investment) * 100
    return roi

Financial Formula: Net Present Value (NPV)

NPV is used to analyze the profitability of a projected investment.

\[ NPV = \sum_{t=1}^{n} \frac{R_t}{(1+i)^t} - C_0 \]
Press 'S' to see these notes. Explain the variables to the stakeholders: - R_t is the net cash inflow during a single period t. - i is the discount rate or return that could be earned in alternative investments. - t is the number of time periods. - C_0 is the initial investment.
